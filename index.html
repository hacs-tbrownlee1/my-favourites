MY FAVOURITE THINGS

C++

C++ is a low-level compiled programming language. Its standard library is very rich in features, and there are countless other libraries written for C++ for extra functionality. Compilers for C++ exist for almost every platform imaginable. Common compilers include:

  • GCC
  • Clang
  • MSVC

I program in C++ not because it is easy, but because it is hard. I like challenges.
